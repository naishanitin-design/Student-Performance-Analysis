# 📊 Student Performance Analysis Dashboard

## 📌 Project Overview
This project presents an interactive Power BI dashboard that analyzes student performance using various academic and demographic factors.

## 📈 Dashboard Features
- 📌 Total Students
- 📌 Average Exam Score
- 📌 Average Attendance
- 📌 Average Study Hours
- 📌 Attendance vs Exam Score
- 📌 Exam Score by Gender
- 📌 Exam Score by Motivation Level
- 📌 Exam Score by Parental Education
- 📌 Exam Score by School Type
- 📌 Exam Score by Teacher Quality
- 📌 Interactive Slicers

## 🛠 Tools & Technologies
- Power BI
- Python
- Pandas
- Microsoft Excel
